# Suraj Ballamudi — Portfolio Website

Personal portfolio site for Suraj Ballamudi, SAP MM & FICO Functional Consultant.

## Tech Stack

- Pure HTML + CSS + Vanilla JS — no framework, no build step
- WebGL lightning shader (hero background)
- WebGL liquid metal shader (About + CTA sections)
- CSS `@property` animated shimmer (all other sections)
- Lenis smooth scroll (CDN)
- Cormorant Garamond + Inter (Google Fonts CDN)

## File Structure

```
/
├── index.html      ← entire site (single file, fully self-contained)
└── README.md
```

## Deployment

Hosted on a static server at [surajballamudi.online](https://surajballamudi.online)

To update: replace `index.html` on the server via FTP/SFTP or your hosting file manager.
